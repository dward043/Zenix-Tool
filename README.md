**ZENIX**, siber güvenlik araştırmacıları ve öğrenciler için geliştirilmiş, terminal tabanlı bir güvenlik test aracıdır. Modern bir arayüze ve hızlı çalışma yapısına sahiptir.

## 🚀 Özellikler

- 🔑 **Brute-Force Attack:** Wordlist kullanarak HTTP POST tabanlı kaba kuvvet saldırısı simülasyonu.
- 🔍 **Vulnerability Scanner:** Hedef IP üzerindeki açık portları tespit eder ve servis bilgisi (banner) toplar.
- 🌊 **DoS Simulator:** Yüksek hızlı UDP paket gönderimi ile ağ dayanıklılık testleri.
- 🖥️ **Modern CLI:** Renkli terminal çıktıları ve estetik ASCII banner.
